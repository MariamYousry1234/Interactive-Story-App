# Interactive Story App

**Description**  
Interactive Story App is a C# Windows Forms application where the story changes based on the userâ€™s choices. Users navigate through **Introduction**, **Beginning**, **Middle**, and **End**, making decisions that lead to different outcomes.

---

## Features
- Story divided into multiple sections using **TabControl**.  
- Dynamic story text updates based on user choices.  
- Multiple decision points with **Buttons**.  
- Reset functionality to restart the story.  

---

## How to Run
1. Open the project in **Visual Studio**.  
2. Build the solution.  
3. Run the application and start the story with the **Start Story** button.  

---

## Technologies
- C#  
- Windows Forms (.NET Framework)
